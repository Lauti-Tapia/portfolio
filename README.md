## 🇪🇸 Español

Un portfolio personal moderno y responsivo construido con Next.js 13+ y TypeScript. Incluye soporte bilingüe (español/inglés), presentaciones interactivas de proyectos y animaciones fluidas.

### ✨ Características

- 🌍 **Soporte Bilingüe** - Cambio fluido entre español e inglés
- 📱 **Completamente Responsivo** - Optimizado para todos los dispositivos (móvil, tablet, desktop)
- 🎨 **UI/UX Moderno** - Diseño limpio con animaciones suaves usando Framer Motion
- ⚡ **Optimizado para Performance** - Construido con Next.js 13+ App Router
- 🎯 **Proyectos Interactivos** - Dashboards de Power BI embebidos y demos en vivo
- 📧 **Integración de Contacto** - Formulario de contacto con Formspree
- 🔍 **SEO Optimizado** - Meta tags y HTML semántico

## 🚀 Getting Started | Cómo Empezar

### Prerequisites | Prerrequisitos
- Node.js 18+ 
- npm or yarn

### Installation | Instalación

1. **Clone the repository | Clonar el repositorio**
   ```bash
   git clone https://github.com/Lauti-Tapia/portfolio.git
   cd portfolio
   ```

2. **Install dependencies | Instalar dependencias**
   ```bash
   npm install
   ```

3. **Run the development server | Ejecutar servidor de desarrollo**
   ```bash
   npm run dev
   ```

4. **Open your browser | Abrir navegador**
   Navigate to | Navegar a [http://localhost:3000](http://localhost:3000)

### Available Scripts | Scripts Disponibles

```bash
npm run dev          # Start development server | Iniciar servidor de desarrollo
npm run build        # Build for production | Construir para producción
npm run start        # Start production server | Iniciar servidor de producción
npm run lint         # Run ESLint | Ejecutar ESLint
```

## 🌍 Internationalization | Internacionalización

The portfolio supports both Spanish and English through React Context API | El portfolio soporta español e inglés a través de React Context API:

- **Language Toggle | Cambio de Idioma** - Click the globe icon in the header | Hacer clic en el ícono del globo en el header
- **Automatic Translation | Traducción Automática** - All content switches seamlessly | Todo el contenido cambia sin problemas
- **SEO Friendly | SEO Optimizado** - Proper meta tags for both languages | Meta tags apropiados para ambos idiomas

## 📱 Responsive Design | Diseño Responsivo

The portfolio is fully responsive with breakpoints | El portfolio es completamente responsivo con breakpoints:
- **Mobile | Móvil:** < 640px
- **Tablet:** 640px - 1024px  
- **Desktop:** > 1024px

## 🎨 Customization | Personalización

### Adding New Projects | Agregar Nuevos Proyectos
1. Update the projects array in `app/page.tsx` | Actualizar el array de proyectos en `app/page.tsx`
2. Add translations in `app/context/LanguageContext.tsx` | Agregar traducciones en `app/context/LanguageContext.tsx`
3. Include project assets in `public/` | Incluir assets del proyecto en `public/`

### Styling | Estilos
- Uses Tailwind CSS utility classes | Usa clases utility de Tailwind CSS
- Custom colors defined in `tailwind.config.js` | Colores personalizados definidos en `tailwind.config.js`
- Animations handled by Framer Motion | Animaciones manejadas por Framer Motion
